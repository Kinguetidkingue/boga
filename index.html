WHAT IS HTML?

HTML is the World Wide Web's core markup language. Originally, HTML was primarily designed as a language for semantically describing scientific documents. It stands for HyperText Markup Language.

HTML was created in 1994 by British computer scientist Tim Berners Lee who worked for the CERN, the European Centre of Particle Acceleration. It's the backbone of web development. Since its creation, HTML has evolved trough many versions and is now replaced by HTML living Standard. This HTML version is maintained by WHATWG group

Learning well HTML can lead you to better understand how to procede with web development. You can interact with the increasing digital world.

WHAT ARE HTML ATTRIBUTES?

HTML attributes are special words used inside the opening tag to control the element's behavior or appearance. They provide additional information about an element and are always included in the opening tag. Attributes usually come in name/value pairs like: name="value".
For example, in the tag <div id="rectangle">, id is the attribute name
and rectangle is the attribute value.

Which Technology Should I Learn First?

The choice of technology to start learning depends on your interests, goals, and the specific field you want to pursue. However, some popular and widely used technologies that are often recommended for beginners include:
  • HTML/CSS: These are the foundational languages for web development and are essential for creating and styling web pages.

  • JavaScript: A versatile programming language used for web development, allowing you to create interactive and dynamic web applications.

  • Python: A beginner-friendly programming language known for its simplicity and readability. It is widely used in various fields, including web development, data analysis, and machine learning.

  • SQL: A language used for managing and manipulating databases. It is essential for working with data and is commonly used in web development and data analysis.

  • Git: A version control system that allows you to track changes in your code and collaborate with others. It is widely used in software development projects.

How Long Does It Take To Create A Static Program?

The time required to create a static program can vary depending on several factors including the complexity of the program, your prior programming experience, and the programming language you choose to use. For beginners with little to no programming experience, it may take anywhere from hours to days or maybe to months to create a simple static program depending your ability to learn fast or not. This could involve learning the basics of the programming language, understanding the syntax, and writing the code for the program. However, with consistent practice and dedication, you can gradually improve your programming skills and create more complex programs over time.
