What Is HTML ?

HTML is a markup language used to create the structure of web pages or web documents. It stands for HyperText Markup Language and is the backbone of web development. HTML has been invented by a British computer scientist, whose name is Sir Tim Berners Lee in 1989 when working for the CERN the European Particle Physics Laboratory based in Geneva. He wrote the first web client and server in 1990. His specifications of URIs, HTTP and HTML were refined as Web technology spread. The first official version was HTML1.0 released in 1994. The World Wide Web is a fast-changing world, so that many versions of HTML including HTML5 are retired. HTML5 is replaced by the current specification known as the HTML Living Standard and is maintained by the Web Hypertext Application Technology Working Group (WHATWG), a consortium of the major browser vendors (Apple, Google, Mozilla, and Microsoft). HTML1.0 version as the beginning has led the foundation of the World Wide Web known as W.W.W.

A. HTML Elements

HTML consists of various elements that define the structure and content of a web page. These elements are represented by tags, which are enclosed in angle brackets.
Building a basic webpage reqiers the use of the following tags:

  1. The DOCUMENT TYPE self-closing tag => <!DOCTYPE HTML>
  2. The HTML pair of opening and closing tags => <html> & </html>
  3. The HEAD pair of opening and closing tags => <head> & </head>
  4. The TITLE pair of opening and closing tags => <title> & </title>
  5. The BODY pair of opening and closing tags => <body> & </body>

HTML tags meaning in a page

  • The DOCTYPE single tag <DOCTYPE html>
    is an instruction to the web browser about which version of HTML the page is written in. It is generally associated with HTML before closing the tag.

  • The HEAD opening tag <head> and the closing tag </head>
    are indicating the head of the document and where the metadata is written. Metadata gives context to data, explaining its origin, nature, structure, and purpose. For example, a dataset of numbers is meaningless without metadata indicating if they are purchase dates, ages, or prices. Metadata can also include information such as the author of a document, the creation date of a photo, or the column names in a spreadsheet

  • The TITLE opening tag <title> and the closing tag </title>
    contain the title of the web page

  • The BODY opening tag <body> and the closing tag </body>
    is the content of the page. There can be headers, text, paragraphs, hyperlinks, forms, photos and/or videos

  • The HTML opening tag <html> and the closing tag </html>
    enclose all the tags and preventing the web browser where is the beginning and the end of a web page

HTML elements are the building blocks of web pages. They define the structure and content of a webpage, allowing developers to create headings, paragraphs, links, images, lists, and more. Each element is represented by a tag, which is enclosed in angle brackets (e.g., <tag>). HTML elements can also have attributes that provide additional information about the element.

B. HTML Attributes

Attributes provide additional information about HTML elements. They are used to specify properties such as the source of an image, the destination of a link, or the style of an element.

C. Progress with HTML

Learning HTML effectively is really important to master your knowledge to the next level. You have to start with the basics to get the taste of what are HTML tags and then gradually build your knowledge that should culminate to a concret project including to building a small-sized website. Here are some steps to help you progress. Be steady in practice !

  1. Understand the basic structure of an HTML document.
  2. Learn about common HTML elements and their usage.
  3. Practice creating simple web pages using HTML.
  4. Explore advanced topics such as forms, tables, and multimedia elements.
  5. Experiment with different attributes to enhance your HTML elements appproach, learning and understanding.

Remember to practice regularly and refer to online resources, tutorials, and documentation to deepen your understanding of HTML. Fortunately there is a huge and diverse amount of resources and documentation you can access and then better practice yourself and/or with a group of students, coders and others.
